URL Shortener Microservice

Project Usage

Add a site's address to the url above to receive a shortened url which will in turn redirect you to your original url



  • Follow the following pattern while typing your original url: base_url/new/{TYPE_SITE_ADDRESS}
  • You will get a JSON response with your original url and a shorter url. Copy the shorter url value and paste it as such: base_url/{SHORTER_URL_VALUE_GOES_HERE}
  • You will be redirected to your original url.
